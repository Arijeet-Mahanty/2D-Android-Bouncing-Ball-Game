# Bouncy Ball 2D 🏀💥
Bouncy Ball is a casual, fast-paced Android mini-game designed to test player reflexes. The goal is simple: balls bounce around the screen, and you must tap them to destroy them as quickly as possible.

# 🎮 Gameplay Features
Reflex-Based Action: Quick-tap mechanics that require precision and speed.

Physics-Based Bouncing: Balls utilize Unity's 2D physics engine for realistic bounces off screen edges.

Mobile Optimized: Lightweight 2D graphics and touch-responsive controls tailored for Android devices.

# 🛠️ Technical Specifications
Engine: Unity 2022.3.46f1 (LTS)

Platform: Android

Language: C#

Physics: Rigidbody2D and CircleCollider2D for interaction.

# 📲 How to Install (Android)

Download the Bouncy_Ball.apk file from the repository.

Transfer the file to your Android device.

Open the file to install (ensure "Install from Unknown Sources" is enabled in settings).

Start tapping and enjoy!
